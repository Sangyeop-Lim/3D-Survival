주요 특징 : 
3인칭 시점에서 플레이어 조작

커피 아이템: 먹으면 이동 속도 빨라짐, 10초 후 리스폰

버섯 아이템: 먹으면 더블 점프 가능, 10초 후 리스폰

점프대와 움직이는 플랫폼으로 높이 올라가기

간단하지만 도전적인 위로 올라가기 플레이

조작법
WASD / 방향키 : 이동

마우스 : 시점 조절

스페이스바 : 점프 / 더블 점프 (버섯 아이템 획득 시)

개발 환경
Unity

C#

Unity Input System
